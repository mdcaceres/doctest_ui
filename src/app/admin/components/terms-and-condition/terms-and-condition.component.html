Frequent questions

How do I create a new project?

To create a new project, go to the project management navigation bar and click on the "Project" button "Create". Fill in the required details such as project name, description, and project client.

How do I create test cases?

To create test cases, navigate to the project's Test Cases section and click on the "Create Test Case" button. Provide the necessary information such as test case title, description, steps, and expected results.

How do I execute test cases?

To execute test cases, open the test case list page and click on the "Execute" button. Select the relevant test environment and provide any additional details. After executing the test case, mark the results as Pass, Fail, or Pending.

How do I create bugs?

To create a bug, navigate to the Project Bug Reporting section and click on the "Create Bug" button. Fill in the bug details, including a concise title, description, and any relevant attachments or screenshots.

How do I view test execution results?

Go to the Reports section of the application. You can view various reports such as test execution summary. Select the desired report type and specify any filters or parameters to generate the report.
